A guide to every button in NovAMPP

Restart Compose - Recreates Docker Compose containers
Reset Compose - Removes all Compose images and recreates them from scratch
Stop Compose - Stops all Compose containers
Start Compose - Starts Compose (automatically done on startup of NovAMPP)
